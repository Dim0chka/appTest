# Тестовое задание: Карусель баннеров

## Цель задания

Необходимо сделать карусель из баннеров, где есть 5 картинок. При этом карусель должна быть замкнутой (бесконечно). Автопрокрутка через каждые 3,5 секунды. В карусели баннеров должен быть виден предыдущий и следующий баннер. Пример прикрепил.
Можно использовать библиотеки для анимации, но нельзя использовать готовый метод слайдера.
